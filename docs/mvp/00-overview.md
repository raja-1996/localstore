# LocalStore — MVP Roadmap

> Each MVP is self-contained and shippable. Work with engineering-manager one MVP at a time.

| MVP | Theme | What It Proves |
|-----|-------|----------------|
| [MVP 1](mvp1-discovery.md) | Discovery | Can users find local merchants? |
| [MVP 2](mvp2-trust.md) | Trust | Can users decide which merchant to pick? |
| [MVP 3](mvp3-engagement.md) | Engagement | Will users interact inside the app? |
| [MVP 4](mvp4-transactions.md) | Transactions | Will users pay in-app? |
| [MVP 5](mvp5-social.md) | Social | Can the community drive growth? |
| [MVP 6](mvp6-intelligence.md) | Intelligence | Can we serve non-tech-savvy users? |

## Dependency Chain

```
MVP 1 (Discovery) → MVP 2 (Trust) → MVP 3 (Engagement) → MVP 4 (Transactions) → MVP 5 (Social) → MVP 6 (Intelligence)
```

Each MVP validates a hypothesis before investing in the next layer.
