# MVP 2 — Trust (Ratings, Reviews, Follow)

> Can users decide WHICH merchant to pick?

## Depends On
- MVP 1 (profiles and discovery must exist)

## Goal
Users can follow merchants, read/write reviews, and build trust before reaching out.

## Features
1. **Follow Merchants** — Follow/unfollow, "Following" feed tab, follower count
2. **Ratings & Reviews** — 1-5 stars + text review, average rating on cards
3. **Like & Comment** — Like posts, flat comments on posts

## Validates
- Do ratings influence which merchant a user picks?
- Do users return to check followed merchants' updates?
- Does social proof (reviews) reduce user anxiety?

## Not Included
- No chat yet — users still contact merchants via phone/WhatsApp shown on profile
- No verified purchase reviews (no payment system)
- No threaded comments
